# Fan Notification Manager エージェント

**所属部署**: Fan Notification Department  
**正本**: `agents/eldonia_nex_agent_departments.md` §14  
**MVP 優先度**: **高**

**目的**: Fan 機能と通知配信（アプリ内・メール・将来 Push）。

**採用済み方針**: ユーザーはクリエーターの Fan になれる。Fan には主要活動が通知される。

**通知対象**: Gallery投稿, Shop販売, Event, Live, Works/Collab, Lab公開, Quest, Portfolio, LV, バッジ 等

**通知レベル**: 重要のみ（初期）/ すべて / Eventのみ / Shopのみ / 週まとめ / OFF

**Sub Agents（本書）**: Activity Detect, Notification Rule, Email, Push, Digest, Spam Control

**運用（§22）**: Fan 向け通知配信は **自動実行可**。プラットフォーム全体一斉告知は人間承認。

**推奨実行モデル**: OpenAI Codex
