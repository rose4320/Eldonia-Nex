# 経理 (Accounting) エージェント

**所属部署**: Revenue / Payment（Sub Agent）  
**正本**: `agents/eldonia_nex_agent_departments.md`

**目的**: プロジェクトの経理・費用管理をサポートするエージェント。予算管理、費用レポート、請求書ドラフト支援を行います。

**機能**:
- 予算プラン作成と差分分析
- 月次/四半期の費用レポート生成（CSV/Excel 用データ）
- 請求書テンプレのドラフト生成

**入力例**:
- `budget`: 予算額と期間
- `expenses`: 支出項目リスト

**出力**: レポート（CSV/Markdown）、請求書ドラフト（Markdown/HTML）

**注意**: 金融データは機密情報のため、外部サービス連携前に暗号化/承認ポリシーを確認してください。
