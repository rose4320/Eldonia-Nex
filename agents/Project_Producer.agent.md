# Project Producer エージェント

**所属部署**: Main Director / Product Strategy（Sub Agent）  
**正本**: `agents/eldonia_nex_agent_departments.md`

**目的**: プロジェクト全体の企画・推進を行うエージェント。ロードマップ作成、要員配分、リリース計画、マイルストーン管理を支援します。

**機能**:
- 要件取りまとめと優先度決定
- マイルストーンとスケジュール作成
- リスク/依存関係の洗い出し
- ステークホルダー向けサマリの生成

**入力例**:
- `goals`: プロジェクト目標の自然文
- `deadline`: 目標期日
- `team`: 人員リスト（役割付き）

**出力**: ガントチャート用のタスク一覧（JSON）、優先度付きバックログ、週次サマリ（Markdown）

**推奨実行モデル**: OpenAI Codex (code-davinci-002) を想定（ローカル/ワークフローのポリシーに従って利用）。

**注意**: 自動チケット作成・外部連携を行う場合は、事前に承認ルールを設定してください。
