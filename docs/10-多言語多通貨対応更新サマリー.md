## 💰 多通貨対応更新

### 対応通貨拡張

| 通貨         | 通貨コード | シンボル | 小数点以下 | 地域           |
| ------------ | ---------- | -------- | ---------- | -------------- |
| 日本円       | `JPY`    | ¥       | 0桁        | 日本           |
| 米ドル       | `USD`    | $        | 2桁        | アメリカ・国際 |
| ユーロ       | `EUR`    | €       | 2桁        | ヨーロッパ     |
| 韓国ウォン   | `KRW`    | ₩       | 0桁        | 韓国           |
| 中国元       | `CNY`    | ¥       | 2桁        | 中国           |
| ベトナムドン | `VND`    | ₫       | 0桁        | ベトナム       |

### 実装機能

- **通貨設定API**: 対応通貨取得・ユーザー通貨設定
- **為替レートAPI**: リアルタイム為替レート取得
- **価格変換API**: 通貨間の自動価格変換
- **決済多通貨対応**: 各通貨での決済処理

## 🔧 技術実装詳細

### API エンドポイント追加

#### 多言語関連

```http
GET /api/localization/languages     # 対応言語一覧
PUT /api/users/me/language          # ユーザー言語設定
GET /api/artworks/{id}?lang=ko      # 多言語コンテンツ取得
```

#### 多通貨関連

```http
GET /api/currencies                 # 対応通貨一覧
GET /api/currencies/rates           # 為替レート取得
GET /api/currencies/convert         # 価格変換
GET /api/users/me/locale            # 地域設定取得
```

### データベース設計影響

#### 新規テーブル追加

- `languages`: 対応言語マスタ
- `currencies`: 対応通貨マスタ
- `exchange_rates`: 為替レート履歴
- `content_translations`: コンテンツ翻訳データ

#### 既存テーブル拡張

```sql
-- ユーザーテーブルに地域設定追加
ALTER TABLE users ADD COLUMN preferred_language VARCHAR(5) DEFAULT 'ja';
ALTER TABLE users ADD COLUMN preferred_currency VARCHAR(3) DEFAULT 'JPY';
ALTER TABLE users ADD COLUMN timezone VARCHAR(50) DEFAULT 'Asia/Tokyo';

-- 作品テーブルに多言語対応
ALTER TABLE artworks ADD COLUMN original_language VARCHAR(5) DEFAULT 'ja';

-- 価格テーブルに通貨情報追加
ALTER TABLE prices ADD COLUMN currency VARCHAR(3) DEFAULT 'JPY';
```
