# Eldonia-Nex 機能要件定義書

## 1. ギャラリー機能

### 1.1 作品投稿・管理機能

#### 1.1.1 作品投稿

**機能概要**: クリエーターが作品をアップロードし、詳細情報を登録する機能

**詳細仕様**:

- **対応ファイル形式**:
  - 画像: JPEG, PNG, GIF, WebP (最大20MB)
  - 動画: MP4, MOV, AVI (最大500MB)
  - 音声: MP3, WAV, FLAC (最大100MB)
  - ドキュメント: PDF (最大50MB)
- **メタデータ入力**:
  - タイトル (必須, 100文字以内)
  - 説明 (任意, 2000文字以内)
  - タグ (最大10個)
  - カテゴリ (必須選択)
  - 価格設定 (無料/有料選択)
  - ライセンス種別
  - 成人向けフラグ

**ユースケース**:

1. クリエーターがログイン後、投稿画面にアクセス
2. ファイルをドラッグ&ドロップまたは選択
3. メタデータを入力
4. プレビュー確認後、投稿実行
5. EXP 100ポイント獲得

**検証条件**:

- ファイルサイズ・形式チェック
- 不適切コンテンツ検出（AI審査）
- 著作権チェック（重複検出）

#### 1.1.2 作品編集・削除

**機能概要**: 投稿済み作品の情報更新・削除機能

**詳細仕様**:

- メタデータ変更（価格変更は制限あり）
- ファイル差し替え（同形式のみ）
- 公開/非公開切り替え
- 削除機能（売上実績がある場合は制限）

### 1.2 作品検索・閲覧機能

#### 1.2.1 検索機能

**機能概要**: 多様な条件で作品を検索する機能

**詳細仕様**:

- **検索方法**:
  - キーワード検索（タイトル・説明・タグ）
  - カテゴリフィルタ
  - 価格帯フィルタ
  - 投稿日期間指定
  - ファイル形式フィルタ
  - 人気度・評価順ソート
- **検索結果表示**:
  - グリッド表示（デフォルト）
  - リスト表示切り替え
  - ページネーション（20件/ページ）
  - 無限スクロール対応

#### 1.2.2 作品詳細表示

**機能概要**: 個別作品の詳細情報表示機能

**詳細仕様**:

- **表示要素**:
  - 作品プレビュー
  - メタデータ表示
  - クリエーター情報
  - いいね数・閲覧数
  - コメント一覧
  - 関連作品推薦

- **インタラクション**:
  - いいね/お気に入り追加
  - SNSシェア機能

  - 不適切コンテンツ報告

| **Business** | ¥10,000 | 企業・団体       | 会社組織での業務レベル |

**Free プラン（¥0/月）**:

- **作品投稿**: 月3件まで
- **ファイルサイズ**: 10MB以下
- **ストレージ**: 100MB

- **作品投稿**: 月20件まで
- **ファイルサイズ**: 50MB以下
- **ストレージ**: 1GB
- **Live配信**: 月5回まで（最大50人）
- **手数料**: 12%
