# Eldonia-Nex UI/UX設計書

## 1. UI/UXデザイン原則

### 1.1 デザインコンセプト

- **テーマ**: 「クリエイティブな表現を支援する直感的なプラットフォーム」

- **キーワード**: モダン、シンプル、アクセシブル、インクルーシブ、インスパイアリング

- **ユーザビリティ**: 3クリック以内で主要機能へアクセス可能
