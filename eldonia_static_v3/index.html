Questとは？

挑戦（Quest）を達成してEXPと信用を獲得。その積み重ねが、あなたの価値と収益につながります。

  • 1. 参加する
    気になるQuestにエントリー
  • 2. 挑戦する
    作品やスキルでミッションに挑戦
  • 3. 評価される
    コミュニティ評価・承認
  • 4. 報酬を得る
    EXP・信用・報酬を獲得

あなたの成長が、すべての価値に変わる

EXP

経験値を貯めるとレベルが上がり、新しい機会を解放します。

信用

評価が高いほど、報酬率やチャンスが増えます。

収益

活動が活発なほど、将来的な収益に繋がります。
